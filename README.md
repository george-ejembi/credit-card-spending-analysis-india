# credit-card-spending-analysis-india
Exploratory and segmented analysis of credit card spending behavior across cities, gender, card types, expense categories, and time.


# Credit Card Spending Analysis in India

# Project Overview
This project presents an exploratory data analysis (EDA) of credit card spending behavior in India. 
The analysis investigates spending patterns across cities, gender, card types, expense categories, 
and time to uncover meaningful consumer insights.

# Objectives
- Analyze city-level spending distribution
- Examine monthly spending trends
- Compare spending behavior by gender
- Identify high-value expense categories
- Evaluate card-type transaction patterns
- Discover high-value customer segments

# Dataset
- Name: Credit Card Spending in India
- Source: Kaggle Data Source
- Records: 52,277,583 transactions
- Attributes: City, Gender, Card Type, Expense Type, Transaction Amount, Date

> ⚠️ The dataset is not included in this repository due to licensing/privacy restrictions.

# Tools & Libraries
- Python 3.10+
- pandas
- matplotlib
- seaborn

# Analysis Highlights
- City-wise total and average spending
- Monthly spending trend analysis
- Gender-based spending comparison
- Expense category distribution
- High-value segment identification (City × Card Type × Expense Type)
